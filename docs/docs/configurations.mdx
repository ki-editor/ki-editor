# Configurations

Ki supports two levels of configurations:

1. **Global**:  
   a. Windows: `%LOCALAPPDATA%\ki\config.json`  
   b. MacOS/Linux: `~/.config/ki/config.json`
2. **Workspace**:  
   a. `./.ki/config.json`

Note: Workspace config overrides Global config.
