# ≡ Eat

## Keymap

<KeymapFallback filename="Eat"/>

In this Momentary Layer (≡), every core movement means:

> Replace everything between the current selection and the \<movement\> selection with the content of the current selection.

For example, `Eat >>` replaces everything after the end of the current selection until the end of the `>>` selection.
